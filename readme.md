# 🚀 Guia de sobrevivencia: Dominio do Terminal (CLI)

 Este repósitorio foi criado como parte da disciplina da **Sistemas Operacionais**, ultilizando o simulador [Terminal Master](https://andersonrmgomes.github.io/terminal-master/) para praticar comandos essenciais de windows e linux.

 ---

## 📖 Introdução
 A interface de linha de comando (CLI) é uma ferramenta poderosa que permite interagir com o núcleo do sistema oparicional. Dominar esses comandos é fudamental para a automoção de tarefa, admistrção de servidores e desenvolvimento de software.

---

## 💻 comandos aprendidos

 Abaixo, apresento a lista  dos pricipais comandos praticados durante o treinamento gamificado:

  | Comando | Funçao | Sistema | Exemplo de uso |
  | :--- |  :--- |  :--- |  :--- |
  | `pwd`| `Exibe o diretório atual (caminho)` | `Linux`| `pwd`|
  | `cd` | `Altera no  diretório de trabalho`| `Ambos`| `cd/home/usuario/documento`|
  | `ls` | `Lista arquivos e pastas` | ` Linux ` | ` ls -la ` |
  | ` dir` | `Lista arquivos e pastas  ` | `Windows ` | `dir /w ` |
  | ` mkdir ` |  ` Cria um novo diretório ` | `Ambos ` | `mkdir nova_pasta ` |
  | ` touch` |  ` Cria um arquivo vazio  ` | ` Linux  ` | ` touch arquivo.txt` |
  | ` type ` |  ` Exibe o conteúdo de arquivos  ` | ` Windows ` | `type arquivo.txt ` |
  | ` del ` | ` Remove arquivos` | `  Windows` | ` del arquivo.txt` | 
  | ` cp` | `  Copia arquivos/diretórios ` | `  Linux ` | `cp arquivo.txt copia.txt` |
  | `move` | ` Move ou renomeia arquivos` | ` Windows ` | `move arquivo.txt pasta/` |
